> Requirements
- A Linux, Windows, or MacOS operating system (32-bit or 64-bit)
- A Java 6 or later runtime environmemt
  (A JDK is required if you wish to execute Java plugins)
- The SWT library
  (downloaded at start-up, requires an Internet connection)
- The Jython library
  (downloaded at start-up, requires an Internet connection)

> How to install?
- Extract the content of this archive to a directory of your choice.
- (Optional) Visit https://www.pnfsoftware.com/download and:
   * download the appropriate version of SWT your system
   * download a Jython package
- Drop these two files in the bin/ directory within your JEB extraction folder

> How to execute?
- On Linux, open a terminal and type: "bash jeb_linux.sh"
- On MacOS, open a terminal and type: "bash jeb_macos.sh"
- On Windows, open a terminal and type: "jeb_wincon.bat"
